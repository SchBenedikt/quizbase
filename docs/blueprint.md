# **App Name**: PulsePoll

## Core Features:

- Poll/Quiz Creation: Presenter can create interactive polls (e.g., multiple choice, open text, ratings) and quizzes with custom questions and options.
- Session Management: Generate a unique code or shareable link for each interactive session. Presenters can start, pause, and advance polls.
- Participant Response Interface: Participants can join a session using a code/link and submit anonymous responses via a simple, mobile-friendly web interface.
- Real-time Result Visualization: Display live poll results instantly to the presenter (and optionally participants) using dynamic visualizations like bar charts for multiple choice or word clouds for open text answers.
- AI Question Assistant: An AI-powered tool that provides suggestions to rephrase poll questions for clarity, engagement, or neutrality based on the input topic or question draft.
- Data Persistence: Securely store poll configurations, participant responses, and session data for historical analysis and retrieval.

## Style Guidelines:

- The design will follow a light scheme. Primary color: A vibrant, expressive magenta (#E01CBB) reflecting dynamic interaction. Background color: A subtle, desaturated lavender-pink (#F9ECF7) to ensure content readability. Accent color: A deeper, rich indigo (#4B0EA8) for strong visual contrast and highlighting key interactive elements.
- Headline and body font: 'Poppins' (sans-serif) for its bold, geometric, and contemporary feel, ensuring clarity and impact for all text.
- Use modern, clean, and abstract icons with subtle gradients or fill effects, aligning with the expressive Material You aesthetic. Icons should adapt to emphasize interaction and real-time updates.
- Dynamic and adaptive layouts that cater to both presenter and participant views on various screen sizes. Employ expressive use of white space and fluid, asymmetric arrangements to create an energetic and engaging presentation experience. Prioritize visual hierarchy for results visualization.
- Implement smooth, responsive transitions for results updates, question changes, and participant actions. Subtle micro-interactions will provide instant feedback, enhancing the real-time, interactive nature of the application.
